# AQA Cypress

Test automation project created for Cypress homework.

## Tech stack

- JavaScript
- Cypress
- Prettier
- Husky
- lint-staged

## Setup

Install dependencies:

```
npm install
```

## Run Cypress

Open Cypress UI:

```
npm run cy:open
```

Run tests in headless mode:

```
npm run cy:run
```

## Format code

Format all files:

```
npm run format
```

Check formatting:

```
npm run format:check
```

## Pre-commit check

Prettier runs automatically before commit for staged files using Husky and lint-staged.
